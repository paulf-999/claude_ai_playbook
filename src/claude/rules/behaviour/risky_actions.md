# 🚨 Rules — Risky actions

Always confirm with the user before taking any of the following actions:

- 🗑️ Deleting files, branches, or data
- ⚠️ `git reset --hard`, `git push --force`, amending published commits
- 💣 Dropping database tables or schemas
- 🔧 Modifying CI/CD pipelines or shared infrastructure
- 🌐 Pushing to remote, opening/closing PRs, posting to external services
