# Style: Data / Report

## Use case
Analytics reviews, metrics reporting, data findings. Dense but structured, number-led.

## Parameters

| Parameter | Value |
|---|---|
| background | `#FFFFFF` |
| header_bar | `false` |
| title_font | Georgia |
| title_size | 32pt |
| title_bold | true |
| title_colour | `#1F3864` |
| body_font | Inter |
| body_size | 18pt |
| body_colour | `#2F2F2F` |
| accent_colour | `#1F3864` |
| accent_secondary | `#FF8F00` (use for data callouts and key numbers) |
| max_bullets | 5 |
| dark_mode | false |
| icons | black stroke |
| title_slide_bg | `#FFFFFF` |
| title_slide_title_colour | `#1F3864` |

## Notes
- Use `accent_secondary` (`#FF8F00`) to highlight key numbers or stat callouts within bullet text where supported.
- Slides may include chart placeholder text where data visualisations are expected.
