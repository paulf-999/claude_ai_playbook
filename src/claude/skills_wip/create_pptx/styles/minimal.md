# Style: Minimal

## Use case
Design-led, editorial, letting content breathe. Typography does the work.

## Parameters

| Parameter | Value |
|---|---|
| background | `#FFFFFF` |
| header_bar | `false` |
| title_font | Georgia |
| title_size | 38pt |
| title_bold | true |
| title_colour | `#000000` |
| body_font | Inter |
| body_size | 20pt |
| body_colour | `#555555` |
| accent_colour | none |
| max_bullets | 3 |
| dark_mode | false |
| icons | black stroke |
| title_slide_bg | `#FFFFFF` |
| title_slide_title_colour | `#000000` |

## Notes
- No decorative elements — no accent bars, dividers, or colour blocks.
- Maximum whitespace: resist the urge to fill slides.
- 3 bullets maximum enforced strictly — if content doesn't fit, split across slides.
