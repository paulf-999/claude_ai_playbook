# Style: Narrative

## Use case
Pitches, change comms, persuasion. Story arc, emotional resonance.

## Parameters

| Parameter | Value |
|---|---|
| background | `#FFFFFF` |
| header_bar | `false` |
| title_font | Georgia |
| title_size | 40pt |
| title_bold | true |
| title_colour | `#880E4F` |
| body_font | Inter |
| body_size | 20pt |
| body_colour | `#2F2F2F` |
| accent_colour | `#880E4F` |
| accent_secondary | `#F9A825` (use for callouts and emphasis) |
| max_bullets | 4 |
| dark_mode | false |
| icons | black stroke |
| title_slide_bg | `#FFFFFF` |
| title_slide_title_colour | `#880E4F` |

## Notes
- Section divider slides (full-slide title, minimal or no body content) are appropriate for narrative decks to mark story transitions.
- Use `accent_secondary` (`#F9A825`) sparingly — for a single callout or emphasis point per slide maximum.
