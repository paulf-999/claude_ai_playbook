# Style: Workshop

## Use case
Facilitation, brainstorming, interactive sessions. Energetic and participatory.

## Parameters

| Parameter | Value |
|---|---|
| background | `#FAFAFA` |
| header_bar | `false` |
| title_font | Georgia |
| title_size | 38pt |
| title_bold | true |
| title_colour | `#7B1FA2` |
| body_font | Inter |
| body_size | 22pt |
| body_colour | `#2F2F2F` |
| accent_colour | `#7B1FA2` |
| max_bullets | 4 |
| dark_mode | false |
| icons | black stroke |
| title_slide_bg | `#FAFAFA` |
| title_slide_title_colour | `#7B1FA2` |

## Notes
- Larger body size (22pt) gives participants more breathing room when reading from a shared screen.
- Slides may include activity prompts or discussion questions as body content.
