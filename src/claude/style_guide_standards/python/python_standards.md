# 🐍 Python Standards - Overview

* Top-level entry point for the team's Python standards
* As a rule of thumb, we follow [PEP 8](https://peps.python.org/pep-0008/) is the baseline
* With the exception of minimal team overrides (described below) taking precedence

---

## 📏 PEP 8 overrides

| Rule | PEP 8 default | Team standard |
|---|---|---|
| Maximum line length | 79 characters | **120 characters** (enforced by ruff and flake8) |

---

For the exhaustive list of the team's Python standards, see [python_style_guide_standards_details.md](./python_standards/python_style_guide_standards_details.md).
